## SPICE Simulation

1. Ensure there is **hspice** in the current environment.
2. Put the basic circuit to `./Circuits/XXX.cir`.
3. Change the parameters in `gates/Circuits/generated/generate_hspice.py`.
4. run `python ./gates/Circuits/main.py`.
